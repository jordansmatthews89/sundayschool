# Landing page — Family Faith

Single-page signup for the newsletter.

## Setup

1. In beehiiv: **Growth → Signup forms** → Create or edit a form → copy the **Embed** code.
2. Open `index.html` and find the div with `id="beehiiv-embed"`.
3. Replace the placeholder inside it with your full beehiiv embed code (script + iframe).
4. Optionally update the Privacy link in the footer.

## Hosting

- Upload `index.html` to any static host (Netlify, Vercel, GitHub Pages, or your domain’s host).
- Or use beehiiv’s **Hosted signup page** and share that URL — no hosting needed.

## Customization

- Edit headline, tagline, and benefits in `index.html`.
- Colors are in the `:root` block; change `--accent` to your brand color (default is a soft green).
