# Christian Curriculum & Bible Studies — For Families

Reusable lesson plans, study guides, and a scope-and-sequence for **families** — at home, in Sunday school, or in family-focused small groups. Designed so parents and kids can do the same lesson together.

## What’s in here

| Folder / file | What it’s for |
|---------------|----------------|
| **scope-and-sequence/** | Year-at-a-glance: which series and topics when. |
| **templates/** | Lesson plan (leader guide), student sheet, and **family guide** (at-home discussion + activity). |
| **lessons/** | Ready-to-use lessons. Each has a leader guide, student sheet, and **family take-home** (discussion + at-home activity). |
| **devotionals/** | Short daily or weekly devotionals; includes **family devotional** template and a sample. |

## Using it with families

- **In class:** Teach with everyone together. Use the “Families” prompts in each lesson so parents and kids can answer the same questions at different depths.
- **At home:** Send the **family take-home** (in each lesson folder) so families can reread the passage, use the discussion questions at dinner or bedtime, and do the at-home activity during the week.
- **Family worship:** Use any lesson or the family devotional template for a short weekly family time (10–15 min).

## Age levels

- **Families:** Discussion questions that work for all ages; one activity that parents and kids do together; optional “go deeper” for adults.
- **Kids:** Shorter focus time, more games and crafts; simple takeaway and one verse.
- **Youth:** Discussion-heavy; tie to their world.
- **Adults:** Deeper discussion, cross-references, application and prayer.

Templates and samples include notes for families and for adapting by age.
