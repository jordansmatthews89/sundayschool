# Lessons

Ready-to-use Bible studies and Sunday school lessons **for families**. Each folder has:

- **Leader guide** — How to teach (with family prompts).
- **Student sheet** — Handout for each person.
- **Family take-home** — Discussion questions + one at-home activity so families can continue during the week.

| Lesson | Passage | Main idea |
|--------|---------|-----------|
| **fruit-of-spirit-01-love** | Gal 5:22–23; 1 John 4:7–12 | Love comes from God; we love because He first loved us. |
| **jesus-calms-the-storm** | Mark 4:35–41 | Jesus has authority over creation and our fears; we can trust Him. |

To add a new lesson: copy the templates (lesson plan, student sheet, family take-home) into a new folder, fill them in, and add the folder name to this table.
