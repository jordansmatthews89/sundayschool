# The AI Edge — Newsletter Business

AI tools & productivity newsletter + digital products. Built to run with minimal touch using AI.

## Project structure

- `setup-guides/` — Step-by-step instructions for beehiiv, Gumroad, referral program
- `content/` — Welcome sequence, Issue #1, product copy
- `landing-page/` — Newsletter signup landing page (static HTML)
- `templates/` — Reusable Claude prompts for weekly content and social posts

## Quick start

1. Complete beehiiv setup using `setup-guides/beehiiv-setup.md`
2. Host the landing page from `landing-page/` (or paste into beehiiv)
3. Use `content/welcome-sequence/` emails in beehiiv automation
4. Set up Gumroad using `setup-guides/gumroad-setup.md` and `content/gumroad-products.md`
